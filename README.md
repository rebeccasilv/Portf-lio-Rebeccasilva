## 💻 Sobre o Projeto

Este é o meu portfólio profissional, desenvolvido para apresentar meus projetos, habilidades técnicas e trajetória como Desenvolvedora Full-Stack. O projeto foi construído com foco em **performance**, **experiência do usuário (UX)** e um design moderno e interativo.

A plataforma é totalmente responsiva e oferece suporte bilíngue completo (Português e Inglês), permitindo que recrutadores e parceiros de diferentes regiões conheçam meu trabalho.

---

## ✨ Funcionalidades Principais

- 🌍 **Suporte Bilíngue**: Alternância instantânea entre Português e Inglês (PT/EN).
- 🌙 **Dark Mode Premium**: Interface moderna com estética focada em tecnologia.
- 📱 **Totalmente Responsivo**: Otimizado para smartphones, tablets e desktops.
- 🎨 **Animações Fluidas**: Microinterações e transições utilizando Framer Motion.
- 📂 **Catálogo de Projetos**: Divisão entre projetos concluídos e em desenvolvimento.
- 📄 **Integração com CV**: Link direto para download/visualização do currículo atualizado.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza as tecnologias mais modernas do ecossistema Web:

- **React 19**: Biblioteca core para construção da interface.
- **Vite**: Build tool extremamente rápida para desenvolvimento moderno.
- **Tailwind CSS**: Framework CSS utilitário para estilização avançada e responsiva.
- **Framer Motion**: Biblioteca de animações para React.
- **Lucide React**: Conjunto de ícones vetoriais modernos e leves.
- **TypeScript**: Superset do JavaScript para garantir robustez e segurança no código.

## 🚀 Como Rodar Localmente

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/rebeccasilv/Portf-lio-Rebeccasilva.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd Portf-lio-Rebeccasilva
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

A aplicação estará disponível em `http://localhost:3000`.
